# TianJian Xing – Data Science Portfolio

This repository contains my personal data science portfolio website, built using a static site generator and deployed via GitHub Pages.

## 🌐 Live Website
https://Xingtia2.github.io

## Overview
This portfolio showcases selected projects in:
- Cloud computing (AWS)
- Data analysis
- Data visualization

Each project includes:
- Problem statement
- Approach
- Key results
- Visual insights

## 🛠 Built With
- Python
- YAML-based content system
- HTML/CSS (static site)
- GitHub Pages

## Structure
- `content/` → project descriptions
- `static/` → images and styles
- `docs/` → generated website (deployment folder)

## Deployment
Built locally using:
```bash
python build.py

Deployed via GitHub Pages (main branch /docs folder).
```
